=== Nocial  Setwork ===
Author: inout
Nocial Setwork is a LAMP based social network platform.
== Specifications ==
Object oriented approach
Simple custom MVC pattern
Included files: PHP, JS, CSS
Database: Mysql
One click install
== Features ==
Follow
Unfollow
Authentication
Private message
Admin dashboard
Users' profile
Users listing
== Install ==
Upload files into your root folder
Specify the database login details
Launch the installer
